Joy - Voice & Vision Assistant

Note: This serves as the product page for Joy where I keep track of all the feature and enhancement updates

Joy is a virtual assistant which comes with both voice and vision capabilities. Originally started with the idea of aiding those with vision challenges, Joy is now generalized and powered by Rasa for its conversation handling and several Machine Learning models developed using TensorFlow and Keras for providing vision related features.

Virtual assistants are all around us(already!) and their number is expected to grow in the coming years. While we all are accustomed to using touch for interacting with devices, having voice as the interface is going to add a lot of value to the existing systems, especially in aiding those with vision challenges through voice and speech.
